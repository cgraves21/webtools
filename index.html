GitHub Account

Adobe Dreamweaver is a common editing tool. It's great for the novice user as it "helps" you along the way...

You'll want to activate a "GitHub Account" from the website. Think of this as your central hub: you can send data to it for storage and pull that storage for web page creation.

Make sure you have the GitHub Desktop to acompany the website account. They function together like a computer and a cloud.
